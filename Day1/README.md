# Day1

This folder contains JavaScript files for Day 1 of the Backend Web Development course.

## Files

- **first.js**: Initial backend code or exercises for Day 1. See comments in the file for details.
- **second.js**: Additional backend code or exercises for Day 1. Refer to the file comments for explanations.

## Usage

Open these files in your code editor to review or run the JavaScript code. You can execute them using Node.js:

```bash
node first.js
node second.js
```

These files are intended for learning and practicing backend development concepts in JavaScript.
